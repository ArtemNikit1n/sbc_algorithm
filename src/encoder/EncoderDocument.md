# Техническая документация дельта-кодировщиков 

---

## 1. Общее описание кодировщиков

- Xdelta — алгоритм дельта-кодирования использующий скользящее окно для вычисления хеша Adler32.

- Gdelta — использует для хеширования не Adler32, а gear hash.

- Ddelta — алгоритм использующий content define chunking (CDC) для разбиения чанка и вычисления хеша.

- EdeltaSpeed — оптимизация ddelta, использующая эвристику: если данные в целевом чанке и родительском совпали, то данные находящиеся поблизости тоже совпадут. (Не добавляет расширенные блоки в словарь)

- EdeltaCompress — то же самое, но добавляет расширенные блоки в словарь

- Zdelta — алгоритм использующий три указателя для умного вычисления смещения copy инструкций.

- ZdeltaHuffman — используется, если нужно сжать результат Zdelta Хаффманом.

## 2. Назначение файлов

| Название                         | Назначение                                                                                                                         |
|------------------------------------|------------------------------------------------------------------------------------------------------------------------------------|
| `ddelta_encoder.rs`                             | Логика ddelta с двумя оптимизациями для edelta `SpeedIsPriority` и `CompressionIsPriority` |
| `zdelta_comprassion_error.rs`                       |          Возможные ошибки работы алгоритма (`ZdeltaCompressionError`, `DataConversionError`, `StorageError`, `MatchEncodingError`)        |
| `zdelta_match_pointers.rs` |   Вся логика для работы с тремя указателями (используются для умного вычисления смещения)    |
